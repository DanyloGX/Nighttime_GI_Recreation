# Nighttime GI Recreation

Collection of GI recreation for Sonic Unleashed night stages


Collection currently includes:

 Faithful(closer to original GI):
  - Windmill Isle Act 1 (ActN_MykonosEvil)
  - Savannah Citadel Act 1 (ActN_AfricaEvil)
  - Savannah Citadel Act 4 (ActN_SubAfrica_03)

 Revamp(tweaked GI):
  - Windmill Isle Act 1 (ActN_MykonosEvil)
  - Arid Sands Act 1/2 (ActN_PetraEvil)

Currently WIP:
 - Jungle Joyride Act 1/3
 - Skyscraper Scamper Act 1/2

Usage:

 - Unpack Original #"StageID".ar.## and delete all .light files
 - Unpack #"StageID".ar.## from repo and move all .light files to Original #"StageID".ar.## 
 - Open stage in HedgeGI Tool, use Environment Table to set up proper environment color in settings tab
 - Now you can bake own GI for stage!

Not for in-game usage - can broke real-time lighting!
